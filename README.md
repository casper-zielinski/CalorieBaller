# CalorieBaller

Eine iOS App zum Tracken der täglichen Kalorienzufuhr und Makronährstoffe.

## Features

- **Wochentag wechseln** – Tage antippen oder links/rechts wischen
- **Kalorienübersicht** – Fortschrittsring zeigt die Kalorien des gewählten Tages
- **Makronährstoffe** – Balken für Protein, Carbs und Fett
- **Gerichtsliste** – Alle Mahlzeiten des Tages auf einen Blick
- **Detail-Ansicht** – Info-Button öffnet ein Sheet mit der vollen Nährwertübersicht eines Gerichts
- **Tipp des Tages** – Wird beim Start per API aus dem Internet geladen
- **Zustand speichern** – Zuletzt gewählter Tag bleibt auch nach App-Neustart erhalten
- **Haptisches Feedback** – Vibration bei Tippen und Wischen
